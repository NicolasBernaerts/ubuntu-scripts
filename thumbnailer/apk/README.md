# Android APK files Thumbnailer #

## Introduction ##

On Android devices, all programs are installed thru some APK archives. Under Gnome Nautilus, 
these APK archives are recognised as a specific mime types and are displayed with some generic text file icon.

This project handles all the steps needed to configure your file manager to display the APK embedded application icon as thumbnail.

This allows you to handle your APK files from your file manager as they appear on your Android device.

## Description & Installation ##

Just download and run :

    ./apk-thumbnailer-install.sh
